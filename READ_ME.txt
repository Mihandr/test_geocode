Описание программы геокодирования v1.0.3

! -Внимание- !
В программе пока не реализован вывод сообщений об ошибке.
При работе с .exe файлом будьте внимательны
Если прогресс-бар перестает заполняться, возможно что программа словила ошибку.
! ---------- !

!!! Обязательные файлы, без них программа работать не будет !!!
--- py.ico
--- Test.csv (Либо любой другой .csv файл при условии что его имя изменено перед началом работы программы (подробнее смотри пункт 2.1))

Для выполнения геокодирования выполните следующие действия:

1. Запустите программу GEO.exe (Возможно наличие приписаной версии программы)
2. По умолчанию имя файла для геокодирования Test.csv если имя вашего файла совпадает, переходите к пункту 3
2.1 Если имя вашего файла иное, нажмите кнопку 'Change' напротив строчки с именем файла и введите имя вашего файла для геокодирования (без указания расширения)
3. По умолчанию ключ API для геокодирования "0bef970e-5f6c-4b90-9ee6-665830d56159" если вы хотите использовать его, переходите к пункту 4
3.1 Если вы хотите использовать свой API ключ, нажмите кнопку 'Change' напротив строчки с API по умолчанию и введите свой ключ API (получить его можно на сайте Yandex)
4. Запустите выполнение программы путем нажатия кнопки 'Start', прогресс выполнения программы будет отображен на прогресс-баре
4.1 Если какие-то данные были введены неправильно, или вы хотите досрочно завершить выполнение программы нажмите кнопку 'Stop'
5. Завершение выполнения программы будет отображено посредством заполненного прогресс-бара и надписи 'Geocoding complete'
6. Результат выполнения программы будет находиться в файле 'result.csv' 

Документация v1.0.3

Программа предназначена для геокодирования объектов недвижимости с помощью сервисов Yandex API.
Результатом являются координаты объекта и его нормализованный адрес.
Импользует метод загрузки csv файла с адресами (Test.csv) и выгрузки такого же csv файла (result.csv).
Предоставляет доступ к пользовательскому интерфейсу для отображения процесса загрузки.

- Функция distanceBetween вычисляет расстояние между двумя точками на карте
- Функция write_csv пишет данные полученые из геокодера в csv файл
- Функция read_csv получает данные из csv файла для дальнейщей обработки
- Функция geo_code основная функция выполнающая геокодирование адреса объекта
- Класс changeData используется для изменения стандартных значений
--- Функции onOk и onCancel обрабатывают события кнопок
- Класс Mywin используется для создания экранной формы
--- Функция InitUI содержит описания для каркаса формы
--- Функции startGEO и stopGEO запускают соответсвенно выполнение и остановку функции OnStart
--- Функция OnStart обрабатывает результат нажатия и запускает выполнение скрипта
--- Функция create_thread создает поток для работы OnStart
--- Функции GetNameFile и GetDataAPI обрабатывают изменения стандартных значений NAME_FILE_TO_GEO и API соответсвенно
